Step-by-step guide: Deselecting an optional field in the blueprint



1. From the plug-in edit page, navigate to field edit (click the pencil link)

2. Make sure the "Required" option is not selected

3. Save

4. On the blueprint order page, both static and programmatically generated optional fields can be deselected. Position the cursor behind the selected field and press the backspace key to remove the selection

5. The field is deselected.
